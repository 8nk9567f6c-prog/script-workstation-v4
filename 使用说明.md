# 编剧工作台 v5.0 — 网页端使用说明

## 访问方式

### 手机端 & 电脑端
直接双击 `index.html` 即可在浏览器中打开。数据已内嵌在文件中，不需要服务器，任何设备均可使用。

### 在线访问
GitHub Pages: [https://8nk9567f6c-prog.github.io/script-workstation-v4/](https://8nk9567f6c-prog.github.io/script-workstation-v4/)

## v5.0 架构

- **单文件架构** — 只有一个 `index.html`，数据直接内嵌为 JavaScript，无 fetch、无 CORS、无断链
- **相对路径** — 所有链接使用相对路径，不依赖特定 GitHub 仓库名
- **完全离线** — 数据内嵌后首次加载即缓存，无网络也能使用全部功能
- **容错设计** — localStorage 损坏不会崩溃，搜索时输入焦点不丢失

## 文件结构

```
web/
├── index.html      ← 唯一应用文件（数据内嵌，双击即用）
├── data.json       ← 规则数据库（简化版，供编程参考）
├── sw.js           ← Service Worker（离线缓存）
├── manifest.json   ← PWA 配置
├── .nojekyll       ← 确保 GitHub Pages 正确服务中文路径
├── 404.html        ← 错误页面自动跳转
├── 使用说明.md
└── 素材库/
    ├── 热梗名梗.md
    ├── 爆款台词.md
    └── 名场面.md
```

## 如何更新数据

1. 编辑 `data.json`
2. 编辑 `index.html` 中对应的内嵌 DATA 对象
3. 提交并推送到 GitHub
